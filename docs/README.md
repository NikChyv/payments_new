# Платежи под контролем — руководства

Гайды по ролям. Каждому участнику нужен только свой файл.

| Роль | Файл | Кому давать |
|------|------|-------------|
| 🧾 **Клиент** | [client.md](client.md) | компаниям, которые оставляют заявки на оплату |
| 💼 **Бухгалтер** | [accountant.md](accountant.md) | сотрудникам, которые проводят платежи |
| ⚙️ **Администратор** | [admin.md](admin.md) | владельцу: завести клиентов, бухгалтеров, выдать ссылки |

## Что это за приложение

Сервис для бухгалтерской аутсорсинговой компании:

- **Клиент** по персональной ссылке (или через Telegram-бота) оставляет заявку на платёж.
- **Бухгалтер** видит единую очередь всех заявок по срокам и проводит платежи.
- Статусы и уведомления обновляются автоматически — и в вебе, и в Telegram.

**Адрес приложения:** https://nikchyv.github.io/payments_new/app/
